---
title: Project Structure
description: Project structure of Xane, a rollup on Mina Protocol with its own AMM.
---

Xane's [GitHub repository](https://github.com/BerzanOrg/xane-rollup) is a monorepo that contains multiple projects as below.

```bash
/
├── client/
├── docs/
├── frontend/
├── packages/
│   └── xane/
└── tests/
```

-   The rollup client of Xane is at [`client/`](https://github.com/BerzanOrg/xane-rollup/tree/main/client).

-   The docs website of Xane is at [`docs/`](https://github.com/BerzanOrg/xane-rollup/tree/main/docs).

-   The frontend of Xane is at [`frontend/`](https://github.com/BerzanOrg/xane-rollup/tree/main/frontend).

-   A package for common use is at [`packages/xane`](https://github.com/BerzanOrg/xane-rollup/tree/main/packages/xane).

-   All tests are at [`xane-rollup/tests/`](https://github.com/BerzanOrg/xane-rollup/tree/main/tests).
